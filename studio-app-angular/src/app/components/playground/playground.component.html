This view is to configure the selected component and add them to Component basket once configuration is complete. These components can then be utilised in the configuring the pages.

Here is how you do it:

Step 1: Drag and drop the component that needs to be configured

Step 2: Use the Configure section and change the dynamics of the Component, apply and test them.

Step 3: Once the changes are complete, add them to Component basket.

Step 4: Continue with other components or Goto component basket to export the code as a zip file
